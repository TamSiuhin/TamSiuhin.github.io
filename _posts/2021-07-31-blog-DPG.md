# Reinforcement Learning -Deterministic Policy Gradient(DPG)

## Main Idea

![](/images/DP1.bmp)

* Use a deterministic policy network (actor): $a = \pi(s;\theta)$
* Use a value network(critic): $q(s,a;w)$
* The critic outputs a scalar that evaluate how good the action $a$ is

## Train value network

* Transition$(s_t,a_t,r_t,s_{t+1})$
*  Value network makes prediction for time $t$

$$
q_t=q(s_t,a_t;w)
$$

* Value network makes prediction for time $t+1$

$$
q_{t+1}=q(s_{t+1},a'_{t+1};w)
$$

​	where $ a_{t+1}'=\pi (s_{t+1}; \theta )$

* TD error

$$
\delta_t=q_t-(r_t+\gamma\cdot q_{t+1})
$$

* Update

$$
w \leftarrow w-\alpha\cdot\delta_t\cdot\frac{\partial q(s_t,a_t;w)}{\partial w}
$$

## Train policy network

**aim:** 

make value $q(s,a;w)$ increase

* Update $\theta$ so that $q(s,a;w)=q(s,\pi(s;\theta);w)$ increases 

* Increase $q(s,a;w)$, where $a=\pi(s;\theta)$

**DPG:**
$$
g=\frac{\partial q(s,\pi(s;\theta);w)}{\partial \theta}=\frac{\partial a}{\partial \theta}\cdot\frac{\partial q(s,a;w)}{\partial a}
$$
**Gradient ascent:**
$$
\theta\leftarrow\theta+\beta\cdot g
$$

## Bootstrapping

* Value network makes prediction for time $t+1$

$$
q_{t+1}=q(s_{t+1},a'_{t+1};w)
$$

​	where $a'_{t+1}=\pi(s_{t+1};\theta)$

* TD error

$$
\delta_t=q_t-(r_t+\gamma\cdot q_{t+1})
$$

**Bootstrapping**: Using one's estimation to update oneself

How to avoid Bootstrapping?  

Computing the **TD target** using different networks

## Target Network

*  Value network makes prediction for time $t$

$$
q_t=q(s_t,a_t;w)
$$

* Target networks make a prediction for time $t+1$

$$
q_{t+1}=q(s_{t+1},a_{t+1}';w^-)
$$

where $$a_{t+1}'=\pi(s_{t+1};\theta^-)$$ , $\pi$ is the Target policy network, $q$ is the Target value network

### Update Target Network

* Set a hyper-parameter $\tau\in(0,1)$
* Update the target networks by weighted averaging

$$
w^-\leftarrow\tau\cdot w+(1-\tau)\cdot w^-\\
\theta^-\leftarrow\tau\cdot\theta+(1-\tau)\cdot \theta^-\\
$$

still related to $w$ and $\theta$

## Summary

* Policy network makes a decision

$$
a=\pi(s;\theta)
$$

* Update policy network by DPG
  $$
  \theta\leftarrow\theta+\beta\cdot\frac{\partial\alpha}{\partial \theta}\cdot\frac{\partial q(s,a;w)}{\partial w}
  $$

* Value network computes 

$$
q=q(s,a;w)
$$

* Target networks $\pi(s;\theta^-)$ and $q(s,a;w^-)$, compute $q_{t+1}$
* TD error:

$$
\delta_t=q_t-(r_t+\gamma\cdot q_{t+1})
$$

* Update value network by TD

$$
w\leftarrow w-\alpha\cdot \delta_t\cdot\frac{\partial q(s,a;w)}{\partial w}
$$
